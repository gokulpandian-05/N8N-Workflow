# 📊 Data Analytics Automation using n8n

## 🔍 Overview

This project automates data analysis using n8n. It performs:

* Data cleaning & preprocessing
* Metrics calculation (sum, avg, min, max)
* Data visualization using QuickChart
* HTML report generation
* Email automation using Gmail

---

## ⚙️ Workflow

CSV/Webhook → Data Cleaning → Metrics → Chart Generation → HTML Report → Email

---

## 📈 Features

* Automated data preprocessing
* Dynamic chart generation (QuickChart API)
* HTML report with "Dear Consumer" format
* Email-ready output

---

## 🚀 How to Use

1. Import `workflow.json` into n8n
2. Configure Gmail credentials
3. Run workflow

---

## 🛠️ Tech Stack

* n8n (workflow automation)
* JavaScript (Function Node)
* QuickChart API (visualization)
* Gmail API (email automation)

---

## 📷 Output

* Chart URL
* HTML report
* Email report
